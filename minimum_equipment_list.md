# Farmerstree Command Center: Minimum Equipment List

운목 대표님의 지시에 따라, Farmerstree 지휘소(디지털 사이니지) 운영을 위한 **최소한의 필수 장비 목록**을 정리하였습니다.

## 1. Core Hardware (지휘소 본체)
- **미니 PC (또는 라즈베리 파이 5)**
    - **용도**: 24시간 사이니지 구동 및 자동화 스크립트 실행
    - **권장 사양**: 
        - CPU: Intel N100 이상 (또는 ARM Cortex-A76)
        - RAM: 8GB 이상 (브라우저 메모리 점유 대비)
        - Storage: 128GB SSD
    - **추천 모델**: Beelink, GMKtec 등 N100 미니 PC (가성비 최적)

## 2. Display (상황판)
- **세로형 모니터 (Vertical Monitor)**
    - **용도**: 뉴스, 날씨, 미션 현황판 출력
    - **사양**: 27인치 이상, FHD (1920x1080) 해상도, IPS 패널 (시야각 확보)
    - **참고**: 피벗(Pivot) 기능이 있는 스탠드 필수, 혹은 베사(VESA) 마운트로 벽걸이 설치

## 3. Network & Peripherals (통신 및 주변기기)
- **무선 키보드/마우스 세트 (소형)**
    - **용도**: 초기 세팅 및 긴급 점검용 (평소엔 숨김)
    - **추천**: 로지텍 K400 Plus (터치패드 일체형)
- **안정적인 Wi-Fi 공유기**
    - **용도**: 클라우드(GitHub)와의 끊김 없는 동기화 및 API 호출

## 4. Software Environment (소프트웨어)
- **OS**: Ubuntu Linux (안정성) 또는 Windows 11 (편의성)
- **Browser**: Chrome (Kiosk Mode 설정)
- **Automation**: Python 3.10+, Git

## 5. Power & Safety (전원 및 안전)
- **스마트 플러그 (IoT)**
    - **용도**: 원격 재부팅 및 전력 모니터링
    - **추천**: Tapo, Hejhome 등
- **멀티탭**: 3구 이상, 개별 스위치형
